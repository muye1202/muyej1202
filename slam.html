Extended Kalman-filter SLAM for Turtlebot

A EKF SLAM pipeline for Turtlebot: a library for robot kinematics and 2D transformation calculation; a wheel odometry library for differential drive robot; a EKF SLAM module coupled with the wheel odometry; and a data association scheme that can keep track of obstacles around the robot.
